## Developed using

- Node.js
- Express.js
- Socket.IO
- PeerJs

### How to use

Install all the required node_modules

npm i

Run Server

Server Started on Port 3310.

*Important :* Allow the camera and audio permissions asked by the browser.

Open browser and goto http://localhost:3310/
